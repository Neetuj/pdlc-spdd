# pdlc-spdd

Structured Prompt-Driven Development for Product Management.

A version-controlled library of reusable AI prompts for product managers,
organized by phase: Discovery, Execute, Deliver.

## Structure

prompts/discovery - understand market, customers, problems
prompts/execute - requirements, roadmap, prioritization
prompts/deliver - launch, GTM, messaging, enablement
templates/prompt-template.md
scripts/new-prompt.sh

## Starter Prompts

| Prompt | Phase |
|--------|-------|

| buyer-persona.md | discovery |
| market-requirements-to-backlog.md | execute |
| roadmap-prioritization.md | execute |

## Create a New Prompt

Run: ./scripts/new-prompt.sh

## Commit Convention

add: new prompt
update: improved prompt
